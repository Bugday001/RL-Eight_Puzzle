# 八数码问题-强化学习
识别手写八数码问题，并使用强化学习求解。(Dueling DQN)

# 使用方法
执行`Get_loction.py`。出现检测界面，在检测界面按`s`即可开始求解八数码。

默认使用img中的图像识别。
若使用摄像头需将`is_video=0`改为`1`。

可以修改`mo_DQN_net.py`中的`difficulty_steps`来修改问题难度
